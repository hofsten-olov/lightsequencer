# Light Sequencer - Unit Definitions File
# =========================================
#
# This file defines DMX fixture types and their channel mappings.
# Each unit definition starts with [UnitName] and lists channel functions.
#
# Format:
#   [Unit Name]
#   channel_number = Function Name
#   channel_number = Function Name
#   ...
#
# Channel numbers are relative offsets from the DMX start address.
# For example, if DMX address is 10 and channel 1 = Mode,
# then Mode is on absolute DMX channel 10.
#
# Lines starting with # are comments and are ignored.
# Blank lines are ignored.
#
# You can add new units by copying an existing block and modifying it.
# =========================================
